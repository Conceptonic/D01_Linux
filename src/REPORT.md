8. Part 8
* Устанавливаю службу с помощью команды sudo apt install openssh-server
* Добавляю службу в автозапуск с помощью команды sudo systemctl enable ssh
* Открываю конфиг файла командой sudo nano /etc/ssh/sshd_config, нахожу #Port 22 и меняю на Port 2022
* Перезапускаю службу sudo systemctl restart ssh
* С помощью команды ps aux | grep sshd проверяю процесс. ps — процессы, a — все пользователи, u — детальная информация, x — системные процессы
* Устанавливаю netstat с помощью команды sudo apt install net-tools
* Смотрю вывод с помощью команды netstat -tan. -t  TCP, -a  все соединения, -n  числовой формат

15. Part 15
* Для выполнения задания был использован планировщик заданий CRON
Команда для открытия редактора заданий:
crontab -e
Добавлена строка:
*/2 * * * * uptime
Для просмотра текущих заданий использована команда:
crontab -l
Для поиска выполнения задания в системных журналах использована команда:
grep CRON /var/log/syslog
Для удаления всех заданий использована команда:
crontab -r
